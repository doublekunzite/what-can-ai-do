Creative Work

Summarize Long Documents

Extract key points from articles, papers, or reports.

  • Turn a 20-page research paper into 5 bullet points
  • Create meeting notes from a transcript
  • Digest a long email thread

Try this: "Summarize this article in 3 paragraphs: [paste text]"

Brainstorm Variations

Generate multiple versions of an idea.

  • 10 headlines for a blog post
  • 5 ways to describe a product feature
  • Different plot twists for a story

Try this: "Give me 5 variations of 'AI helps with productivity'"

Code Assistance

Help with programming tasks.

  • Explain a confusing function
  • Debug an error message
  • Convert code from one language to another

Try this: "Explain this Python code line by line: [paste code]"

Analytical Tasks

Data Pattern Recognition

Find patterns in unstructured data.

  • Sort survey responses by sentiment
  • Extract names and dates from messy notes
  • Categorize customer feedback

Comparative Analysis

Compare options systematically.

  • Pros and cons of two products
  • Compare three business strategies
  • Weigh different decision factors

Communication

Rewrite for Different Audiences

Adapt tone and complexity.

  • Explain quantum computing to a 10-year-old
  • Make technical documentation user-friendly
  • Formalize casual notes

Translation & Localization

Translate with context (better than Google Translate for nuance).

  • Email translation with cultural context
  • Localize a product description
  • Explain idioms and slang
